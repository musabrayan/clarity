## Quick Commands

```bash
# Development Setup
npm install                 # Install frontend deps
pip install -r requirements.txt  # Install backend deps

# Development Run
npm run dev                 # Frontend (http://localhost:5173)
python app.py              # Backend (http://localhost:3000)

# Production Build
npm run build              # Frontend (creates dist/)
gunicorn -w 4 -b 0.0.0.0:5000 app:app  # Backend

# Testing
npm test                   # Frontend tests
pytest                     # Backend tests

# Docker
docker build -t clarity-backend .
docker run -p 5000:5000 clarity-backend
```
